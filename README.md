# Homebrew Tap

Homebrew formulae for 3000-2 projects.

## Installation

```bash
brew install 3000-2/tap/fid
```

Or:

```bash
brew tap 3000-2/tap
brew install fid
```

## Formulae

| Formula | Description |
|---------|-------------|
| [fid](https://github.com/3000-2/fid) | Terminal UI for viewing git diffs with style |
